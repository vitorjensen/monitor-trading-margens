# Monitor de Preços e Margens de Arbitragem - Trading & Mobilidade

Este projeto automatiza a coleta de preços de mercado de commodities/combustíveis e calcula as margens de arbitragem frente aos custos industriais internos da companhia, servindo como ferramenta de governança de custos e suporte à decisão para a mesa de Trading.

## Tech Stack
* **Python** (Pandas e NumPy para processamento de dados)
* **Playwright** (Web Scraping automatizado dos indicadores de mercado)
